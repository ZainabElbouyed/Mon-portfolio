Description

This project aims to optimize the performance of race cars by designing a self-adjusting rear wing. Using fluid mechanics principles, a system based on Arduino, servomotors, and sensors will dynamically adjust the wing's angle according to the track: flat on straight sections to minimize drag and inclined in corners to maximize grip. A simulation and an experiment will assess the effectiveness of this system, which could provide a better balance between speed and stability in racing.

Description

The SmartBin project aims to revolutionize urban waste management through intelligent and connected trash bins. By integrating IoT technologies, sensors, and optimization algorithms, it enables automated bin opening, real-time monitoring via a connected network, and optimized collection routes through a dedicated application. This system enhances operational efficiency, reduces costs, and minimizes environmental impact by preventing overflows and lowering the carbon footprint of waste collection routes.
