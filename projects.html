My Projects

Explore a variety of projects I've worked on, including web development, IoT systems, and AI-powered applications.

Aerodynamic Optimization: The Rear Wing as a Key to Performance in Motorsport.

This project focuses on optimizing the aerodynamic performance of racing cars through an IoT-based system that dynamically adjusts the rear wing's angle. Using real-time data from sensors monitoring speed, acceleration, and steering angle, the system automatically modifies the wing’s angle to enhance downforce during cornering and reduce drag on straightaways. The result is improved vehicle handling and performance on the track, with the system offering a smart, efficient solution to aerodynamic challenges in motorsport.

Arduino UNO, Potentiometer, Servo Motor.

AirSense: Intelligent Air Quality Monitoring in School Environments.

AirSense is an intelligent solution that uses AI, IoT, and energy-efficient sensors to monitor air quality in schools. Connected sensors continuously track CO₂, temperature, and humidity levels in real-time, while also minimizing energy consumption. The system analyzes the data to detect anomalies and generate alerts for harmful pollution levels. A mobile app allows users to monitor air quality and receive real-time alerts. It dynamically adjusts ventilation to optimize energy use, reduces the school's carbon footprint, and promotes environmental awareness among students.

Raspberry Pi, Arduino UNO, Temperature and Humidity Sensor, CO₂ Sensor, Express.js, Next.js, Python, HTML, JavaScript, CSS, Bootstrap.

EasyCrowd: Intelligent Crowd Management System.

EasyCrowd is transforming crowd management for the 2030 World Cup by leveraging AI, geolocation, and IoT technologies. Using smart sensors, the system monitors real-time movement, detects overcrowded zones, and anticipates potential risks. An intelligent algorithm generates personalized routes and sends automatic alerts to both visitors and organizers, ensuring smooth circulation, optimal coordination, and a safe, seamless experience for spectators and staff. The solution reduces congestion through predictive routing, enables real-time monitoring with IoT and heat maps, and facilitates fluid evacuations through smart guidance.

AI cameras, IR sensors, Pressure sensors, Geolocation technologies, Python, Express.js (Node.js), React.js (JavaScript, HTML, CSS), MongoDB.

GreenTech SmartBin : A Connected Solution for Sustainable Waste Management.

SmartBin is an innovative IoT-based waste management system that uses sensors and real-time connectivity to monitor fill levels, detect presence, and automate bin operations. A dedicated mobile and web app provides live monitoring, sends alerts, and uses smart algorithms to optimize collection routes and scheduling.

ESP32, Ultrasonic Sensor, Servo Motor,IA cameras, Python, HTML, CSS, JavaScript
